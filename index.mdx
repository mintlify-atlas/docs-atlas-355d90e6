The Wikisource extension enhances MediaWiki-based Wikisource installations with powerful tools for transcribing scanned texts and exporting works in multiple formats. It integrates OCR engines for automated text extraction, provides one-click export to EPUB, MOBI, and PDF, and connects works to their Wikibase edition data.

## Getting started

<Steps>
  <Step title="Install the extension">
    Clone the repository into your MediaWiki `extensions/` directory and add `wfLoadExtension( 'Wikisource' );` to `LocalSettings.php`. See [Installation](/installation) for full details.
  </Step>
  <Step title="Configure export and OCR URLs">
    Set `$wgWikisourceWsExportUrl` and `$wgWikisourceOcrUrl` in your `LocalSettings.php` to point to the WS Export and OCR services. See [Configuration](/configuration/settings).
  </Step>
  <Step title="Enable OCR for editors">
    Set `$wgWikisourceEnableOcr = true;`. Editors with the WikiEditor beta toolbar will see the OCR tool in the ProofreadPage editor automatically.
  </Step>
  <Step title="Start transcribing">
    Open any Page namespace article in the editor and click the **Extract text** button to run OCR on the scanned image. See the [OCR feature guide](/features/ocr) for engine selection and language options.
  </Step>
</Steps>

<Note>
  This extension requires MediaWiki 1.46 or later and the [ProofreadPage](https://www.mediawiki.org/wiki/Extension:ProofreadPage) extension. Wikibase integration also requires the Wikibase Client extension.
</Note>
